# Command: /pr:build-media-list

## Description
Research and build a tiered, scored, prioritized contact list for the announcement — including journalists, influencers, podcasters, newsletter authors, industry analysts, bloggers, and community moderators as appropriate.

## Instructions

1. Ask for:
   - Announcement topic and one-sentence summary of the news
   - Target audience profile (who do we want to reach?)
   - Geographic scope (local / regional / national / international)
   - Preferred coverage types (breaking news / feature / trade / podcast / influencer / newsletter / analyst / community — or "all relevant")
   - Any known priority outlets, journalists, or influencers
   - Any contact types to exclude for this campaign

2. Based on the announcement and audience profile, recommend the right contact type mix before building — e.g., "For a consumer product launch I'd recommend 40% journalists, 40% influencers, 20% newsletter authors. Does that fit your goals?"

3. Use web search to identify relevant contacts across all applicable types and beats.

4. Apply the full contact intelligence framework from `pr-journalist-intel.md` including the expanded contact type taxonomy and type-specific scoring criteria.

5. Produce the full tiered contact list with Tier A deep-dives.

6. Flag any coverage gaps — important contact types, beats, or platforms where strong fits couldn't be found.

## Output
Full structured contact list organized by tier and contact type. Save as a file in the working folder. Include a contact type breakdown summary. Offer to proceed to pitch drafting next.
