# Command: /pr:campaign-brief

## Description
Compile all campaign work into a single professional document. The master deliverable of the plugin.

## Instructions

1. Ask: "Is this a pre-campaign brief, mid-campaign update, or post-campaign report?"

2. Based on the stage, load the appropriate sections from `pr-campaign-brief.md`:
   - **Pre-campaign**: Sections 1-7 (excludes coverage results and post-campaign analysis)
   - **Mid-campaign**: Sections 1-7 plus a coverage-to-date snapshot
   - **Post-campaign**: All sections (1-10)

3. Pull from all work already produced in this session or available in the workspace folder:
   - News value assessment output
   - Competitive positioning brief
   - Press release review / draft
   - Media list
   - Pitch copy
   - Outreach execution plan
   - Coverage tracker output (if post-campaign)

4. Note any sections that cannot be completed due to missing information. List specifically what is needed and from whom.

5. Produce the complete brief as a `.docx` file using the document skill. Apply professional formatting with clear section headers, tables, and an executive summary on page 1.

6. Save to the working folder and present a summary of what's included.

## Output
Complete campaign brief as a `.docx` file. Present a file summary and note any incomplete sections.
