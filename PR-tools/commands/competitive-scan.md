# Command: /pr:competitive-scan

## Description
Research comparable announcements and sharpen positioning by identifying what makes this announcement genuinely unique.

## Instructions

1. Ask for: announcement topic/category, organization name, any known competitors or comparable organizations.

2. Use web search to identify 3-5 comparable recent announcements (last 6-12 months) in the same space.

3. Apply the competitive analysis framework from `pr-competitive-analysis.md`.

4. Produce the full competitive brief with USP hierarchy, proof points, and positioning statement.

5. If no strong differentiator is found, say so explicitly and suggest what additional facts or angles might create one.

## Output
Competitive positioning brief. Offer to proceed to release review or journalist intel next.
