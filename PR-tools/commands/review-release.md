# Command: /pr:review-release

## Description
Audit an existing press release draft or produce a new one from scratch.

## Instructions

**If reviewing an existing release:**
1. Ask the user to provide the press release text (paste or point to a file in the workspace).
2. Ask: "What are the 3-5 key messages this release must communicate?"
3. Apply the full review framework from `pr-release-review.md`.
4. Produce: assessment, checklist results, key messages audit, revision recommendations, and revised headline/lead options.

**If drafting a new release:**
1. Ask for: announcement facts (who, what, when, where, why), key messages, spokesperson name/title/quote, boilerplate.
2. Apply all structural standards from `pr-release-review.md`.
3. Produce a complete, pitch-ready press release draft.
4. Apply self-review before delivering — flag any structural gaps.

## Output
Reviewed release with tracked recommendations, or complete new draft. Offer to proceed to journalist intel or pitch drafting.
