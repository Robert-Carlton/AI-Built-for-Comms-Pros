# Command: /pr:subject-lines

## Description
Generate and optimize subject line options for pitch emails. Can be run standalone or as part of pitch drafting.

## Instructions

1. Ask for: announcement summary (1-2 sentences), target journalist type (business / consumer / trade / local / broadcast), and any subject lines already drafted.

2. Generate 8-10 subject line options across these categories:
   - **Data-led**: Leads with a specific number or statistic
   - **Outcome-led**: Leads with the result or impact
   - **News-hook**: Pure news statement, no framing
   - **Question**: Poses the story as a question (use sparingly)
   - **Localized**: Emphasizes geographic relevance

3. For each option, note:
   - Character count
   - Approach type
   - Best suited for (which journalist segment)

4. Apply quality filters: flag anything over 70 characters, anything using forbidden phrases ("press release," "exciting," "revolutionary," "game-changing," "just checking in"), anything that starts with the organization name.

5. Recommend top 3 with rationale.

## Output
Ranked subject line list with character counts, approach labels, and recommendations. Ask if the user wants to proceed to full pitch drafting.
