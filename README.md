## What is PR Tools?

A complete public relations toolkit that runs entirely within Claude Cowork — no Python environment, no API keys, no local dependencies. Install once, [customize with your org's details](https://github.com/Robert-Carlton/AI-Built-for-Comms-Pros/wiki/Customizing-PR-Tools-for-Your-Organization-or-Client), and every press release campaign runs through a structured, professional workflow.

The plugin encodes hard-won expertise across the full PR campaign lifecycle. What are some of the powerful things it can do:   [assess whether a story is actually newsworthy](https://github.com/Robert-Carlton/AI-Built-for-Comms-Pros/wiki/The-tool-commands#pr-pitchingassess-news-value),  [position against competitive announcements](https://github.com/Robert-Carlton/AI-Built-for-Comms-Pros/wiki/The-tool-commands#pr-pitchingcompetitive-scan),  [build a media list](https://github.com/Robert-Carlton/AI-Built-for-Comms-Pros/wiki/The-tool-commands#pr-pitchingbuild-media-list) that prioritizes quality over volume,  [write pitches](https://github.com/Robert-Carlton/AI-Built-for-Comms-Pros/wiki/The-tool-commands#pr-pitchingdraft-pitch) that journalists actually read, and [report results](https://github.com/Robert-Carlton/AI-Built-for-Comms-Pros/wiki/The-tool-commands#pr-pitchingcoverage-tracker) in a way that makes sense to leadership. 

PR Tools can even ["learn"](https://github.com/Robert-Carlton/AI-Built-for-Comms-Pros/wiki/The-Style-Learning-Workflow:-How-to-have-PR-Tools-learn-your-style!) your writing and formatting style! And check out these [tips](https://github.com/Robert-Carlton/AI-Built-for-Comms-Pros/wiki/The-Style-Learning-Workflow:-How-to-have-PR-Tools-learn-your-style!#pr-tools-tips-for-style-learning) for how PR Tools Style Learning capability can supercharge productivity for comms consultants, agencies and corporate comms teams!

Check out the full [toolset](https://github.com/Robert-Carlton/AI-Built-for-Comms-Pros/wiki/The-tool-commands) to enhance and streamline your workflow!
